ftplib_quit

Close connection to server.

SYNOPSIS

#include <ftplib.h>
void ftplib_quit(netbuf *nbp);

PARAMETERS

nbp
A handle returned by ftplib_connect().

DESCRIPTION

ftplib_quit() closes the connection to the remote server and frees any resources associated with the connection.

RETURN VALUE

None.
