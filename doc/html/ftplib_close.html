ftplib_close

Close a data connection.

SYNOPSIS

#include <ftplib.h>
int ftplib_close(netbuf *rnbp);

PARAMETERS

rnbp
A handle returned by ftplib_access().

DESCRIPTION

Closes the data connection specified by 'rnbp' and frees associated resources.

RETURN VALUE

Returns 1 if successful or 0 on error.
