ftplib_cdup

Change to parent directory.

SYNOPSIS

#include <ftplib.h>
int ftplib_cdup(netbuf *nbp);

PARAMETERS

nbp
A handle returned by ftplib_connect().

DESCRIPTION

ftplib_cdup() sends a CDUP command to the remote server.

RETURN VALUE

Returns 1 if successful or 0 on error.
