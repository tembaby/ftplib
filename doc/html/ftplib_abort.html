ftplib_abort

SYNOPSIS

#include <ftplib.h>

int ftplib_abort(netbuf *nbp);

PARAMETERS

A handle returned by ftplib_connect()

DESCRIPTION

Instructs the server to abort previous service command and any associated trnasfers of data.

RETURN VALUE

Returns 1 if successful of 0 on error.
